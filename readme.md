                              HOW TO USE THIS


1. Each project folder contains its own:
   - index.html (main page)
   - css/style.css (styling)
   - js/script.js (functionality)
   - assets/ (images and resources)

2. To view a project:
   - Open the index.html file in your web browser
   - Or use a live server extension for better development experience

3. To explore the code:
   - Open the project folder in a text editor or IDE
   - Review the HTML, CSS, and JavaScript files
   - See the progression of my coding style and abilities
